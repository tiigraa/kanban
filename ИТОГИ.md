## Как проверить:

### Проверка фильтра:
1. Откройте "Активный спринт"
2. Выберите "Иванов" в фильтре
3. Увидите только задачи Иванова

### Проверка перетаскивания:
1. Откройте "Активный спринт"
2. Наведите на любую задачу
3. Зажмите левую кнопку мыши
4. Перетащите в другую колонку
5. Отпустите кнопку

### Проверка создания задачи:
1. Откройте "Панель администратора"
2. Заполните все поля:
   - Заголовок: Тестовая задача
   - Автор: Ваше имя
   - Исполнитель: любой из списка
   - Время: 5
   - Описание: Тестовое описание
   - ID: TE-1234
3. Нажмите "Создать задачу"
4. Вас перенаправит на страницу спринта
5. Новая задача появится в колонке "К выполнению"
